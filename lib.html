backup¶

inpho.lib.backup.data(filename=None, remote_file=None)[source]¶

Backs up the InPhO database, excluding all user information.

Equivilent to: mysqldump $LOGIN –no-data > $OUTFILE mysqldump $LOGIN –tables entity groups idea idea_instance_of idea_link_to institution journal journal_abbr nationality ontotree profession school_of_thought searchpatterns sep_areas sepentries thinker thinker_has_influenced thinker_has_nationality thinker_has_profession work >> $OUTFILE

inpho.lib.backup.graph(filename=None, remote_file=None)[source]¶

Backs up the most recent InPhO graph. This may be better done by just symlinking the daily data mining runs generated by inpho/corpus/sep.py, but since we may include multiple corpora, this will be necessary to write.
